## 🛠️ 최근 작업 내역

### 1. 상품 마케팅 포트폴리오 다차원 진단 기능 고도화 (v1.2.0)
* **백분위 기준 정량적 임계치 로직**: 고정 절대수치 대신 최근 데이터 기반 상대적 백분위를 적용하여 오분류를 최소화했습니다.
  * *노출 과다*: 클릭수 상위 15% 이내 & CVR 브랜드 평균 미만 (광고 비효율 상태).
  * *개선 필요*: 클릭수 상위 20% 이내 & CVR 하위 30% 미만 (상세페이지/옵션 병목 상태).
* **지표 요약 스코어카드 레이아웃 (KPI Card UI)**: 최상단에 노출 과다 상품 수, 개선 필요 상품 수, 금주 실행 필요 액션 수, 고성과 자산 상품 수를 독립된 카드 UI로 표출하여 진입 시 현황 인지 시간을 단축했습니다.
* **마케팅 액션 달성률 게이지 (Semi-donut Chart)**: 제안된 조치의 실행 여부를 체크하면 실시간으로 달성률을 반원형 게이지로 시각화해 피드백 루프를 강화했습니다.
* **Rounded Bar Chart 성과 랭킹 보드**: 카테고리 내 상품들의 유입량 순위에 둥근 코너(cornerradius)를 가미한 바 차트를 렌더링하고, 개선 필요 및 노출 과다 상품은 경고 하이라이트 색상(Red/Orange)으로 강조했습니다.

### 2. 상품 마케팅 포트폴리오 다차원 진단 기능 추가 (v1.1.0)
* **4사분면 (BCG Matrix) 자동 포지셔닝 분석**: 카테고리 내 상품들의 클릭수 및 구매전환율 **중위수(Median)**를 기준으로 상품을 4사분면으로 분류.
* **그룹별 맞춤 마케팅 가이드라인 제안**: 각 사분면에 속한 상품 목록과 해당 그룹의 상태 진단, 그리고 매출 극대화를 위한 구체적이고 즉시 실천 가능한 마케팅 액션 플랜 제공.
* **마케팅 진단 보고서 내보내기**: 생성된 분류 요약 및 상세 마케팅 제안 가이드를 마크다운(.md) 파일로 즉시 다운로드할 수 있는 원클릭 버튼 제공.

### 3. API 인증 정보 관리 체계 개선 (보안 강화)
* **로컬/배포 하이브리드 로드 지원**: 로컬 환경에서는 `.env` 및 `.streamlit/secrets.toml`을 통해 API 인증 키(`NAVER_CLIENT_ID`, `NAVER_CLIENT_SECRET`)를 관리하고, 배포 후에는 Streamlit Cloud의 **Secrets**에서 자동으로 가져오도록 코드를 수정했습니다.
* **수동 입력 지원**: 인증 키가 세팅되지 않은 경우, 사이드바에서 사용자가 직접 키를 수동 입력하여 대시보드를 사용할 수 있게 폴백 기능을 구현했습니다.

### 4. 자동 커밋 및 푸시 훅(Hook) 시스템 구축
* **Git Post-Commit Hook 설정**: 로컬에서 커밋 성공 시 자동으로 원격 저장소에 `push`하도록 `.git/hooks/post-commit`을 작성했습니다.
* **실시간 변경사항 감시 데몬**: 로컬 파일 시스템 변경을 감시하여 변경사항이 저장되는 즉시 자동으로 커밋 및 푸시를 수행하는 `auto_git_sync.py` 스크립트를 가동했습니다.

### 5. 통계 분석 및 시각화 기능 오류 수정
* **Pandas Styler 의존성 해결**: EDA 요약본 제공 시 테이블 내 그라데이션이 `matplotlib` 라이브러리 부재로 인해 로드되지 않던 에러를 해결하고 `requirements.txt`에 명시했습니다.
